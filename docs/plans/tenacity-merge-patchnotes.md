## **综合更新**
- Buff 事件 **“坚定”** 已正式并入 **“坚韧”**，后续不会再单独出现。
- **“坚韧”** 现在同时提供击退抗性与低血线保命能力：
  - 新增 `60%` 击退抵抗
  - 生命值低于 `40%` 时，在 `1.25` 秒内回复 `45%` 最大生命值
  - 内置冷却为 `6` 秒
- 原“坚定”的“受到击退时叠加减伤并回血”机制已移除，以降低场景波动和强度尖峰。

## **错误修复**
- 修复了“坚定”并入“坚韧”后，底层仍保留双 ID 判定的问题。
- 现在 Buff 效果只会由 **“坚韧 / Tenacity”** 触发，不再存在旧 ID 的兼容生效路径。
